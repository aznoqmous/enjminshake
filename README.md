# aznoqmous/enjminshake

Enjmin CPP gameplay programming project using a modern c++ sandbox.

Default features from sandbox :
- hot reloadable shaders
- imgui
- sfml
- eastl

## Added features :
- [x] Entity
  - [x] Sprite animation (idle, walking, hurt, dead)
  - [x] Player
    - [x] Camera follows player 
    - [x] Weapons
      - [x] MuzzleFire
      - [x] Recoil
      - [ ] Screenshake
  - [x] Foes
- [x] Bullets
  - [ ] Entity collisions
    - [ ] Time freeze
  - [ ] Wall collisions 
- [ ] Reskin walls
- [ ] Controller support
